# Streamlit Sentiment Analysis App

A **user-friendly Streamlit web application** to perform **sentiment analysis** on text data. Users can input text or upload datasets to analyze sentiment as **positive, negative, or neutral**. The app leverages Python libraries such as **pandas, scikit-learn, and NLTK** to preprocess data and build predictive models.

---

## Features

- Real-time sentiment prediction for user input  
- Dataset upload for bulk sentiment analysis  
- Interactive and visually appealing Streamlit interface  
- Easy to deploy and extend for other NLP tasks  

---

## Tech Stack

- **Python**  
- **Streamlit**  
- **Pandas, Numpy, Scikit-learn, NLTK**  
- **GitHub** for version control
